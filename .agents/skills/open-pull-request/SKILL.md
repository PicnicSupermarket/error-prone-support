# Open Pull Request

Use this skill when asked to create, open, or prepare a pull request.

* Read [`.github/instructions/pull-request.instructions.md`][pull-request] for the PR
  conventions (title, label, milestone, reviewers, description).
* Read [`.github/instructions/commit-message.instructions.md`][commit-message] for the
  commit message conventions (the PR title and description depend on
  these).
* After preparing the PR, review it against the applicable sections of
  [`.github/instructions/review.instructions.md`][review].
