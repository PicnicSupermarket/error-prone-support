# Write Documentation

Use this skill for requests to add or change `.md` files.

* Read [`.github/instructions/documentation.instructions.md`][documentation]
  for the writing conventions.
* When editing instruction files specifically, closely follow the guidelines in
  [`.github/instructions/instructions.instructions.md`][instructions].
* After implementation, review your changes against the applicable sections of
  [`.github/instructions/review.instructions.md`][review].
* Finally, follow the workflow in `AGENT.md`.
