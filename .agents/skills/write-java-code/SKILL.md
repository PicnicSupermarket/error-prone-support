# Write Java Code

Use this skill when working on Java code outside `*/refasterrules/*`.

* Read [`.github/instructions/java-style.instructions.md`][java-style] for code
  style conventions.
* Read [`.github/instructions/testing.instructions.md`][testing] for testing
  patterns.
* After implementation, review your changes against the applicable sections of
  [`.github/instructions/review.instructions.md`][review].
* Finally, follow the workflow in [`AGENT.md`][agent-md].
